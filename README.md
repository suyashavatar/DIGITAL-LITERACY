Project Overview

This project is a Digital Literacy Portfolio developed as part of the CSE0001 course at VIT Bhopal University. The aim of this project is to understand and apply essential digital skills required for academic and professional growth.


As a Student Digital Ambassador, I explored different areas of digital literacy such as creating digital content, building a professional online presence, using coding and collaboration platforms, practicing professional communication, and understanding cyber safety.


All tasks are organized in this GitHub repository to demonstrate proper digital documentation and presentation skills.

Objectives 

To understand the concept and importance of digital literacy

To create a professional digital identity

To explore coding and collaboration platforms

To develop effective digital communication skills

To understand cyber threats and prevention methods
